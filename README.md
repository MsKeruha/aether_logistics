# Aether Logistics BPM 🌌

**Aether Logistics** — це інтелектуальна веб-система управління бізнес-процесами сервісного центру в умовах розподілених замовлень. Проект розроблений як MVP для демонстрації сучасних підходів до автоматизації сервісного обслуговування.

## ✨ Основні можливості

- **Управління замовленнями**: Повний цикл від реєстрації пристрою до видачі клієнту.
- **Розподілена логістика**: Система обліку замовлень, що надходять з різних джерел/філій.
- **Моніторинг навантаження**: Візуалізація зайнятості майстрів та ефективності центру.
- **Aether AI Diagnostics**: (Концепт) автоматичний аналіз несправностей.
- **Преміальний інтерфейс**: Футуристичний дизайн у стилі Glassmorphism.

## 🛠 Технологічний стек

### Backend
- **FastAPI**: Високопродуктивний API фреймворк.
- **SQLAlchemy**: ORM для гнучкої роботи з БД.
- **JWT Auth**: Надійна автентифікація та ролевий доступ (Admin/Tech/Client).

### Frontend
- **React + Vite**: Швидка розробка та збірка.
- **Tailwind CSS**: Сучасна стилізація.
- **Lucide Icons**: Набір футуристичних іконок.

## 🚀 Швидкий старт

### Використання Docker

```bash
docker-compose up --build
```

### Локальна розробка

#### Backend
1. `cd backend`
2. `pip install -r requirements.txt`
3. `python scripts/seed.py` (ініціалізація БД)
4. `uvicorn app.main:app --reload`

#### Frontend
1. `cd frontend`
2. `npm install`
3. `npm run dev`
